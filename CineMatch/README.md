# CineMatch — Neural Collaborative Filtering Movie Recommender

A movie recommendation system built with a neural collaborative filtering
approach in TensorFlow/Keras. The model learns latent embeddings for
users, movies, and genres, and combines them through a small feed-forward
network to predict user ratings and generate personalized Top-N movie
recommendations.

## Overview

- **Task:** predict how a user would rate a movie they haven't seen, and
  recommend the highest-predicted titles.
- **Approach:** learned embeddings for `user_id`, `movie_id`, and
  `movie_genre`, concatenated and passed through dense layers to output a
  predicted rating.
- **Framework:** TensorFlow / Keras (Functional API).
- **Dataset:** [MovieLens 100K](https://grouplens.org/datasets/movielens/100k/)
  — 100,000 ratings from 943 users on 1,682 movies, collected by the
  GroupLens Research Group, University of Minnesota.

## Model Architecture

```
user_id     --> Embedding(16) --\
movie_id    --> Embedding(16) ---+--> Concatenate --> Dense(32, relu) --> Dense(16, relu) --> Dense(1)
movie_genre --> Embedding(4)  --/
```

Trained with mean squared error loss against observed ratings (Adam
optimizer). Recommendations for a given user are generated by scoring
every unrated movie and ranking by predicted rating.

## Project Structure

```
data/                          movies.csv, ratings.csv
scripts/
  make_sample_data.py            generates a synthetic placeholder dataset
  download_movielens.py          downloads the real MovieLens 100K dataset
recommend.py                    builds, trains, and evaluates the model;
                                 prints Top-5 recommendations for a user
requirements.txt
```

## Setup & Usage

```bash
pip install -r requirements.txt

# Get data (choose one):
python scripts/download_movielens.py    # real MovieLens 100K data
python scripts/make_sample_data.py      # synthetic placeholder data

# Train and generate recommendations:
python recommend.py
```

## Dataset Attribution

This project uses the **MovieLens 100K** dataset, collected by the
GroupLens Research Group at the University of Minnesota.

> F. Maxwell Harper and Joseph A. Konstan. 2015. The MovieLens Datasets:
> History and Context. *ACM Transactions on Interactive Intelligent
> Systems (TiiS)* 5, 4, Article 19 (December 2015), 19 pages.
> https://doi.org/10.1145/2827872

Data is downloaded directly from GroupLens's servers at runtime
(`scripts/download_movielens.py`) rather than redistributed with this
repository, per the dataset's usage terms.
